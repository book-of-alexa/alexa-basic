# Ready for developing using ASK CLI

Written by: Tomoyuki Tochihira

## ASK CLIとは

ここではスキル開発を効率化するためのASK CLI（Alexa Skills Kit Command Line Interface）の使い方について説明します。

ASK CLIとは、コマンドラインからスキル開発に必要なリソースの操作が行えるツールです。例えば対話モデルや公開情報の更新、スキルの申請が行えます。また、ブラウザからの面倒な更新が不要になり対話モデルや公開情報をファイルで管理することが出来るようになります。

Lambdaの作成、更新にも対応しています。スキルのエンドポイントがLambdaの場合には、プログラムソースも一緒に管理することができます。スキルに関連するリソースを一箇所で管理することができて大変便利です。

スキル開発を積極的に行っていく読者ならば、ASK CLIの知識を身に着けておくことで開発スピードが格段に上がります。この章では、ASK CLIの中でもスキル開発によく使うコマンドについて説明します。
